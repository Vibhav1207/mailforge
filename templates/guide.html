Setup Guide

How to Get a Gmail App Password

You need 2-Step Verification enabled on your Google Account before you can create an App Password.
1

Enable 2-Step Verification

Go to myaccount.google.com/security → Click "2-Step Verification" → Follow the setup.

2

Go to App Passwords

Visit myaccount.google.com/apppasswords or search for "App Passwords" in your Google Account settings.

3

Generate a Password

Enter a name like MailForge → Click "Create". Google will show you a 16-character password. Copy it immediately — you won't see it again.

4

Use in MailForge

Paste the 16-character app password in the "App Password" field when sending emails. Do not use your regular Gmail password.

How to Format Your Excel / CSV

Your file must have an email column. Optionally add a name column and any other columns for placeholders.

name email team
Vibhav Patel vibhav@example.com Alpha
John Doe john@example.com Beta
Jane Smith jane@example.com Alpha
Column names are case-insensitive. "Email", "EMAIL", "email" all work.

Using Placeholders in Templates

Use {% raw %}{{column_name}}{% endraw %} in your template body. Each placeholder will be replaced with the corresponding value from each row.

Example Template Body

Dear {% raw %}{{name}}{% endraw %}, Congratulations! You have been selected for team {% raw %}{{team}}{% endraw %}. Best regards, The MailForge Team

Attaching Images Per Recipient

1

Name images after recipients

If your Excel has a "name" column with value Vibhav Patel, name the image file Vibhav Patel.png or Vibhav_Patel.jpg.

2

Upload via the "Upload Images" page or inline

You can pre-upload a folder of images, or select them directly when sending.

3

Default attachment

If no matching image is found for a recipient, the "Default Attachment" (if provided) will be used instead.
